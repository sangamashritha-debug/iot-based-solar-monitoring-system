# iot-based-solar-monitoring-system
An IoT-based Solar Monitoring System using ESP32 for real-time monitoring, solar tracking, dust detection, automated cleaning, and cloud monitoring.
## Project Overview

This project is an IoT-based Solar Monitoring System designed to monitor the health and performance of a solar panel in real time using an ESP32 microcontroller.

The system collects different environmental and electrical parameters through sensors and displays the data for monitoring purposes.

The project also includes advanced automation features such as solar tracking, automatic panel cleaning, damage detection, and cloud monitoring.

---

## Objectives

- Monitor solar panel voltage
- Monitor current
- Measure power generation
- Measure panel temperature
- Measure ambient temperature
- Measure humidity
- Detect dust on the solar panel
- Automatically clean the panel using a wiper
- Track the sun using servo motors
- Detect panel damage
- Send sensor data to the cloud

---

## Hardware Components

- ESP32
- INA219 Voltage and Current Sensor
- DHT22 Temperature and Humidity Sensor
- LDR Sensors
- Servo Motor
- Dust Sensor
- Rain Sensor (Optional)
- Solar Panel
- Battery
- OLED Display (Optional)
